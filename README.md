#  Digital Classroom Timetable

A web-based **Weekly Schedule Management System** built as a UI Design project. It provides dual-role views for students and faculty, allowing interactive timetable management with class assignment, swapping, and monthly reporting.

##  Live Preview

Open `index.html` in any modern browser — no server or installation required.

---

##  Features

- **Student View** — Browse the weekly timetable with subject, teacher, room, and online class details
- **Faculty View** — Add, edit, and manage class schedules
- **Class Swap** — Drag-and-drop style click-to-swap for rearranging classes
- **Online Class Badges** — Visual indicators for virtual classes with meeting links
- **Monthly Report** — Switch to a report page for monthly schedule summaries
- **Color-coded Subjects** — Each subject styled with distinct colors for quick identification
- **Responsive Layout** — Clean, accessible UI built with vanilla HTML, CSS, and JavaScript

##  Usage

### Student View
- See the full weekly timetable at a glance
- Click on a class cell to view details (subject, teacher, room, online link)

### Faculty View
- Click **+ Assign Class** to add a new class to any slot
- Click **⇄ Swap Classes** to enter swap mode — select two cells to swap their contents
- Click any cell to edit or remove a class
